# 🚚🐦 Camión y Pájaro (Unity)

Juego arcade 2D en **Unity** con mecánicas de movimiento lateral, bombas y objetivo final.

## 🚀 Tecnologías
- Unity (C#)

## ✨ Funcionalidades
- Movimiento de personajes
- Lanzamiento de bombas
- Sistema de impactos/vida
- Escena final al completar objetivo

## ▶️ Ejecución
Abrir el proyecto en Unity Hub y ejecutar la escena principal.
